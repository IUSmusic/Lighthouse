# Lighthouse Music Studio

Lighthouse Music Studio is a conceptual coastal music production retreat centered on a lighthouse tower and a private walled compound.

The project combines professional studio use with artist accommodation in a single destination environment. The main lighthouse contains a glass-walled studio / rehearsal floor with panoramic sea views, a resting lounge and washrooms directly beneath the studio floor, and an upper view port / break room. Around the lighthouse, inside the compound, independent guest houses provide private accommodation for artists, producers, and collaborators.

The design is intended as an architectural and creative destination rather than only a recording room. Its purpose is to bring together music production, retreat, recovery, and visual identity in one landmark setting.

## Core Program
- Glass-walled studio / rehearsal floor inside the lighthouse
- Resting lounge and washrooms beneath the studio floor
- View port / break room above
- Independent guest houses within the compound
- Secure, immersive coastal setting

## Design Intent
- Create a distinctive landmark-based music studio concept
- Combine production, hospitality, and retreat functions
- Use the surrounding sea view as part of the creative experience
- Offer privacy, focus, and accommodation for extended stays

## Status
This project is currently presented as an early-stage concept design with initial planning intent and approximate sizing.
